# Проект "Доска объявлений" 

##  Описание проекта
Backend-часть приложения для размещения объявлений. Пользователи могут регистрироваться, просматривать объявления, создавать новые, оставлять комментарии и редактировать свой профиль. Реализована система ролей (USER/ADMIN) и разграничение прав доступа.

##  Используемые технологии и библиотеки

### Основной стек:
- Java 11 — язык разработки
- Spring Boot 2.7.15 — основной фреймворк
- Spring Web — создание REST API
- Spring Data JPA — работа с базой данных
- Spring Security — аутентификация и авторизация
- PostgreSQL — база данных
- Liquibase — миграции базы данных
- Maven — сборка проекта

### Дополнительные библиотеки:
- Lombok — сокращение шаблонного кода (геттеры, сеттеры, конструкторы)
- SpringDoc OpenAPI (Swagger) — документация API

## Архитектура приложения
Проект построен по многослойной архитектуре:
- Controller — REST-контроллеры (обработка HTTP-запросов)
- Service — бизнес-логика
- Repository — взаимодействие с БД через JPA
- Entity — сущности базы данных
- DTO — объекты для передачи данных
- Mapper — преобразование Entity в DTO и обратно

## Безопасность
- Аутентификация через HTTP Basic
- Пароли шифруются с помощью BCryptPasswordEncoder
- Разграничение прав: USER может редактировать только свои объявления и комментарии, ADMIN может редактировать всё
- JdbcUserDetailsManager с кастомными SQL-запросами для работы с существующей схемой БД

## Документация API
После запуска приложения документация Swagger доступна по адресу:
http://localhost:8080/swagger-ui.html

## Запуск проекта
1. Установить PostgreSQL
2. Создать базу данных
3. Настроить параметры подключения в application.properties
4. Создать папки для хранения изображений вручную resources/static/images/ads и resources/static/images/avatars
5. Запустить приложение через HomeworkApplication.main()
